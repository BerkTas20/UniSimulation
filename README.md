# UniversitySimulation
University simulation with python



-A university came to you and said:
  asked you to set up a system that registers students, shows students, deletes students, and organizes students.
  1-The system that will record the recorded information of the student's name, surname, faculty, department, number, type of education and status.
  2-Deleting students with id (unique ID).
  3-Changing student information.
  4-See students information.



-Bir üniversite size geldi ve dedi ki: 
  Öğrenci kaydı yapan, öğrencileri gösteren, öğrencileri silen ve öğrencileri düzenleyen bir sistem kurmanızı istedi.Şu bilgileri de verdi:
  1-Kaydedilen bilgilerde öğrencinin adı,soyadı,fakültesi,bölümü,numarası,öğrenim türü ve durumunu kaydedecek olan sistem.
  2-Öğrencileri id(benzersiz kimlik) ile silme.
  3-Öğrenci bilgilerini değiştirme.
  4-Öğrencileri gör.

I used the SQLITE database.
SQLİTE databesesini kullandım.
